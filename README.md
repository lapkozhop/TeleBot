# Телеграм бот для записи на консалтинговые услуги

Этот телеграм бот предназначен для управления записями на консалтинговые услуги. Пользователи могут создавать, просматривать, редактировать и удалять свои записи через интуитивно понятный интерфейс.

## Функциональность

- **Запись на услугу** - создание новой записи с указанием темы и контактных данных
- **Просмотр записи** - отображение текущей активной записи
- **Редактирование записи** - изменение существующей записи
- **Удаление записи** - полное удаление текущей записи
- **Дашборд** - открытие веб-дашборда для аналитики

## Установка и настройка

### Требования

- Python 3.7+
- Библиотека `pyTeleBotAPI`

### Установка зависимостей

```bash
pip install pyTeleBotAPI
```

### Настройка бота

1. Создайте бота через [BotFather](https://t.me/BotFather) в Telegram
2. Получите API токен
3. Замените токен в коде:
```python
bot = telebot.TeleBot("YOUR_BOT_TOKEN_HERE")
```

## Запуск

```bash
python bot.py
```

## Использование

После запуска бота отправьте команду `/start` для инициализации. Бот предоставит меню с следующими опциями:

### Основное меню

- **Записаться на услугу** - создать новую запись
- **Ваша запись** - просмотреть текущую запись
- **Изменить запись** - меню редактирования/удаления записи
- **Дашборд** - открыть веб-дашборд (требуется запущенный локальный сервер)

### Меню редактирования

- **Редактировать** - изменить существующую запись
- **Удалить** - удалить запись (с подтверждением)

## Структура данных

Бот хранит данные пользователя в глобальной переменной `user_data`, которая содержит:
- Тему консультации
- Контактные данные пользователя

## Особенности

- Интуитивно понятный интерфейс с inline-кнопками
- Подтверждение при удалении записей
- Проверка наличия данных перед редактированием
- Интеграция с веб-дашбордом

## Разработка

Проект построен на библиотеке `pyTeleBotAPI` и использует:
- Inline-клавиатуры для навигации
- Обработчики следующих шагов для ввода данных
- Глобальное хранение состояния пользователя

## Примечания

- Для работы функции "Дашборд" необходимо запустить это проект https://github.com/lapkozhop/Dashboard_lab
- В текущей реализации данные хранятся в памяти и сбрасываются при перезапуске бота
- Для production использования рекомендуется добавить постоянное хранилище данных
